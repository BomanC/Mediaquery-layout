Mitä media queryt ovat?

Media queryt ovat sääntöjä, joiden mukaan selaimet toteuttavat eri CSS-sääntöjä eri kokoisilla näytöillä. Ne toimivat hieman samantapisesti kuin if..else-lauseet ohjelmoinnissa.

Breakpointit

Jos selainikkuna on 600 kuvapistettä tai pienempi, taustaväri on vaaleanharmaa:

@media only screen and (max-width: 600px) {
  body {
    background-color: lightgrey;
  }
}
